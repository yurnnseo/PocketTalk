## 네트워크프로그래밍 2인 프로젝트 (PocketTalk💬)

### ✔️ 기능
 * 로그인
 * 프로필 수정 (이름, 상태메시지, 프로필 사진 변경)
 * 개인, 단체 채팅
 * 상대방에게 이모티콘/사진 전송
 * 채팅창 내에 공지 등록
 * 1:1 미니게임 (사과 게임)

### 컨셉:메타몽(포켓몬)
### 코드 간단 설명
1. clientMain:로그인 화면
->이름,주소,포트번호 텍스트 필드만 코드에 넣고 나머지는 이미지에 넣어 붙이기.

2. clientView:개인/단체 채팅창
->메시지 보내기 기능은 있음. 사진/이모티콘 보내기 기능 추가, 게임시작버튼 추가

3. 새로운 clinetjava파일:친구/채팅 목록 확인 창
-> 내 프로필 확인, 수정, 채팅 초대 등의 기능 넣기

4. minigame:사과게임 화면

5. server:클라이언트와의 연결창
